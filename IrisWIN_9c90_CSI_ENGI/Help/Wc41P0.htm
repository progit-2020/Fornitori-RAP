Interrogazioni di servizio

Introduzione

Questa scheda consente di effettuare estrazioni particolari dalla base dati IrisWIN che non sono previste nelle normali funzioni dell'applicativo.

Riferimenti sul database

T002_QUERYPERSONALIZZATE

Esegui

Permette di eseguire la selezione desiderata; se la query viene eseguita correttamente verranno visualizzati i dati nella griglia, altrimenti verrà segnalato errore.

Nome

Permette di assegnare un nome ad una query che si intende salvare o di richiamare una query salvata in precedenza tramite la lista a scorrimento. E' possibile filtrare le query da rendere disponibili tramite il Filtro dizionario

Griglia variabili

Al momento del caricamento di una query, propone l'elenco delle variabili presenti nella query permettendo di inserire i valori e il tipo dei filtri da applicare

Variabile

Nome della variabile presente nella query

Tipo

Tipo del valore della variabile presente nella query può assumere i valori:

  • Stringa: Il valore da assegnare alla variabile è un carattere generico alfanumerico (Es.: ROSSI)
  • Data: Il valore da assegnare alla variabile è una data nel formato gg/mm/aaaa (Es.: 21/03/2008)
  • Numero: Il valore da assegnare alla variabile è un numero che può o meno contenere cifre decimali (Es.: 12,54 oppure 12)
  • Sostituzione: Il valore da assegnare alla variabile è un carattere alfanumerico che verrà sostituito nel testo dalla query ed eseguito insieme ad esso (Es.: AND MATRICOLA = '3')

    Valore

    Valore da assegnare alla variabile presente nella query

Dettaglio scheda

Griglia

Riporta il risultato ottenuto dall'estrazione eseguita.

E' possibile selezionare una o più righe tenendo premuto il tasto Ctrl per poter eventualmente copiare solo parte dell'estrazione in Excel o in un altro file. Sono infatti disponibili le seguenti funzioni premendo col tasto destro del mouse:

Copia su Excel

Copia le righe selezionate negli appunti, in formato testo. I dati sono separati con le tabulazioni e possono essere incollati in Excel, riportando lo stesso incolonnamento della griglia.
