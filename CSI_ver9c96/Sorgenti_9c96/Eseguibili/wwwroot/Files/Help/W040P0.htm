Upload documenti

Introduzione

Questa scheda permette di gestire l'upload dei documenti da pubblicare per i dipendenti.

E' possibile selezionare la cartella di destinazione e verificare la corretta sintassi del nome del file da caricare.

Dettaglio scheda

Tipologia documento

Consente di selezionare la tipologia di documento per cui si desidera caricare il file.

Selezionare la cartella di destinazione

Lo schema ad albero nella sezione a sinistra elenca la struttura di cartelle della tipologia selezionata.

Nella casella a destra vengono elencati i file contenuti nella cartella selezionata.

Per procedere è necessario selezionare la cartella in cui si desidera caricare il file.

File da caricare

E' necessario selezionare il file da caricare utilizzando il file Sfoglia... e quindi procedere al caricamento effettivo premendo Upload.

L'applicativo verifica la corretta sintassi del file e l'estensione, visualizzando un riepilogo delle informazioni prima dell'effettivo upload.
